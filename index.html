Where it lives

  • HQ Brain → Google Drive (Shawn's "EZO Marketing - HQ" folder)
  • DG Brain → GitHub + a folder on Clay's Mac (~/Documents/EZO-DG-Brain)
  • Always backed up & version-controlled — nothing gets lost

How to access & use it

  • Easiest: Claude Desktop → Cowork, point it at the folder
  • In Projects: upload EZO-BRAIN-INDEX.md into any project
  • Then just ask in plain English — "Who's the AssetSonar buyer?"

How you know it's working

Ask a project an EZO-specific question (e.g. an ICP or a target).
✅ Working = it answers with our real strategy from the brain
❌ Not connected = a vague/generic answer
If generic → the brain file isn't loaded in that project.

How it stays in sync

  • HQ docs → auto-pull to your computer nightly
  • DG toolkit → saved to GitHub; team gets a Slack alert when it changes
  • Projects: re-upload a file when Slack says it changed
