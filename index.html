How to Use It
Four steps, every session.

The brain works best when you follow these steps in order. Skip step 3 and you'll get a blank session — the most common mistake.

1

Get Claude Pro and get added

Claude Projects require a Pro account ($20/mo). Ping Clay to be added to the DG Brain. Ping Shawn to be added to Marketing HQ. Once added, the project appears in your left sidebar under Projects — no install required.

💡 Don't have Pro? Ping Clay — the team may cover it.
2

Open from inside the Project — not the homepage

Log in to claude.ai → click Projects in the left sidebar → click EZO DG Brain → start a new conversation from there. This loads all the project files and memory automatically.

⚠️ Opening from the homepage or a new chat creates a blank session with zero context.
3

Paste the session opener at the start

Every session, paste the opener prompt first. It tells the brain to reload context before you start asking questions. Takes 5 seconds and saves you 10 minutes of re-briefing.

💡 Bookmark the opener prompt — you'll use it every time.
4

Ask like a teammate, not a search engine

Don't ask "what is demand gen?" Ask "what should our Q3 ITSM campaign focus on given our current pipeline?" The more context you give, the better the answer. The brain already knows the basics — push into the specifics.

💡 If the answer feels generic, add more context about what you're actually trying to do.
⚠️

The #1 mistake: opening from the homepage. Always go Projects → EZO DG Brain → New conversation. If Claude asks "what do you work on?" at the start of a session, you're in the wrong place — close it and start from the Project.

📋 Session opener — paste this every time before your first question
"You are the EZO DG Brain. Start by reviewing the project files to refresh your context, then confirm what you know about current campaigns, our brand voice, and any open priorities. What's top of mind?"

If You Hit Roadblocks
Common issues and exactly how to fix them.

Most problems come down to one of five things. Here's how to diagnose and fix each one.

🚨 The brain doesn't know who we are

Claude is responding generically. It doesn't seem to know our ICPs, brand voice, or what we do. Answers feel like they came from a search engine.

✅ Fix: You're in the wrong session

Close this chat. Go back to claude.ai → Projects → EZO DG Brain → start a new conversation from inside the project. Then paste the session opener prompt. Never start from the homepage.

🚨 Context seems stale or outdated

The brain is referencing old campaigns, wrong priorities, or information that's been superseded. It's not up to date.

✅ Fix: The project files need updating

Ping Clay or Shawn — a project file needs to be updated. The brain only knows what's in its files. If something changed and the file wasn't updated, the brain doesn't know about it.

🚨 HubSpot data isn't loading

You asked for pipeline data or CRM metrics and got a message that live data is unavailable, or got numbers that don't look right.

✅ Fix: Check the HubSpot connection

The HubSpot integration may need to be refreshed. Ping Clay with what you asked for — he can pull the data directly or re-authorize the connection. This is occasional, not a daily issue.

🚨 The answer is good but too long / too short

You got a wall of text when you wanted a quick answer, or a one-liner when you needed a full brief.

✅ Fix: Tell it the format you need

Add format instructions to your prompt: "Give me this in bullet points," "Keep it under 200 words," "Give me a full brief with sections." The brain takes direction well — just be explicit about what you want.

🚨 I don't see the project in my sidebar

You logged in but don't see EZO DG Brain or Marketing HQ anywhere in Claude.

✅ Fix: You haven't been added yet (or need Pro)

Two things to check: (1) Do you have Claude Pro? Projects are Pro-only — ping Clay if unsure. (2) Have you been added to the project? Ping Clay for DG Brain access or Shawn for Marketing HQ. Once added, it appears within a few minutes.
